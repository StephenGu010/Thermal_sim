# 改动说明

## 2026-05-21 双 Profile Outline 增强

用户需求：对比 HOLOSUN 风格热成像 Outline 效果和当前工程实际效果，判断可见光摄像头是否可实现，并改进工程以同时支持 Tiny1-C 热成像主链路和可见光演示链路。

本次改动：

- `core/outline_processing.py` 增加 `Thermal Tiny1-C` 与 `Visible Demo` 双 Profile。
- Thermal Outline 增加坏点抑制、时域 EMA、热目标 gating、边缘密度上限和强弱边缘亮度分层。
- Visible Demo 单独使用可见光灰度、bilateral/Gaussian 降噪、Sobel/Scharr、Canny-like linking、小连通域清理和边缘密度上限。
- `ui/main_window.py` 增加 `Profile` 下拉框；Visible Demo 下关闭热目标分类和热点标记。
- `core/scope_renderer.py` 在 HUD 显示 `THERMAL RAW` 或 `VISIBLE DEMO`。
- 更新 `README.md`、`docs/holosun_outline_reverse_engineering.md`，新增 `docs/工程说明.md`。

验证：

- 已运行 `.venv/bin/python -m compileall -q main.py core ui`。
- 已用合成 Mock raw14 调用 Thermal/Visible Outline，确认输出 shape/dtype 正常。

注意：

- `Visible Demo` 只能展示黑底亮边风格，不能写成真实热成像增强。
- Tiny1-C 实测应优先确认状态栏为 `mode=Y16(raw)`；若为 `8bit(fallback)`，细节会明显退化。
- `output/last_source.json` 是运行状态文件，本次没有把它纳入代码改动设计。
